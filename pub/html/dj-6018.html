찾아오시는길

  • 찾아오시는길
    • 주소 : 대전광역시 유성구 현충원로 251(갑동 산 23-1)
    • 전화번호 : 042) 718-7114
      ※ 설날,한식,현충일,추석등은 묘소를 참배하시는 분이 많아 교통이 혼잡하니 대중교통을 이용하여 주시기 바랍니다.
  • 자가용으로 오시는길
    • 경부/호남/대전 - 통영/대전 - 당진 고속도로/대전 남부순환도로
      • 호남고속도로 유성 IC에서 나와 32번 국도를 이용하여 계룡산(동학사) 방면으로 3km 진행 우측 소재
    • 논산/공주방향에서 오실 때 1번, 32번 국도 이용
      • 계룡산(동학사) 입구에서 유성방면으로 5km 좌측 소재
  • 지하철로 오시는길
    • 대전역에서 반석행 지하철 탑승
      • 대전역 서광장 방향 출구 이용
    • 현충원역 하차
      • 에스컬레이터 또는 엘리베이터 이용하여 2번 출구(보훈모시미 이용), 3번 출구(시내버스)로 이동
    • 현충원역 2번출구 앞에서 보훈모시미 차량 탑승
      • 보훈모시미승강장 이용 * 사회적 거리두기 3~4단계시 운영중단
  • 운행시간표(승강장 출발시간)
    테이블
    주요역·터미널 이용방법(소요시간)
    대전역 버스 107번 : 대전역 동광장 → 현충원(총 50분) 지하철 : 대전역 → 현충원역 이동 후 버스 환승(총 50분)
    서대전역 버스 612번 : 서대전역 → 서대전네거리역 하차하여 지하철로 현충원역 이동 후 버스 환승(총 60분)
    동부고속·시외버스터미널 버스 102번 : 현충원역 이동 후 버스 환승(총 60분)
    ※ 토·일, 공휴일은 현충원 정문까지 운행
    서부 시외버스터미널 버스 113번 : 현충원역 이동 후 버스 환승(총 60분)
    유성 시외버스터미널 유성 시외버스정류장에서 버스 107번 현충원 하차(총 20분)
    유성 고속버스터미널 도보로 유성시외버스터미널까지 이동(10분) 후 버스 107번·48번 : 현충원 하차(총 20분)
    지하철 현충원역
    ※ 버스 환승시 이용    		 참배객모시미(셔틀버스) : 2번 출구 → 현충원(5분)
    버스 107번·48번 : 3번 출구 → 현충원(5분)
    ※ 토·일, 공휴일은 102번 이용 현충원 정문 하차
    • * 102번은 휴일(토·일, 공휴일 포함)만 현충원(정문) 경유
    • * 에스컬레이터 또는 엘리베이터 이용하여 3번 출구로 이동하여 버스 이용
